Insertion sort

Insertion sort is a simple sorting algorithm. We start checking from the second number in the list and place it before the first number if it is smaller than the first number (Considering we are sorting in ascending order). Then we move to the next number and “insert” it into the position which makes this sub-list of length 3 sorted. Then we move to the next number and insert it into position in the same manner which makes this sub-list sorted. We continue this process till we reach the last element and insert it into the position and we end up with a sorted list.

Time Complexity

Worst Case: O(n2)

Average Case: O(n2)

Best Case: O(n)
