Merge sort

Merge sort is a divide and conquer algorithm which divides the input list into two halves and calls merge sort on them and merges the returned lists to end up with a sorted list. The division results in n lists containing one number each. The first two of these lists are merged such that the resulting list containing two elements is a sorted list. The next two single element lists are merged into a sorted list of length two. Now these two lists of length two are merged resulting in a sorted list of length four. Same process occurs for the next four elements and now we get two sorted lists of length four which are merged to get a sorted list of length. This process continues till the whole list is merged into a sorted list.

Time Complexity

Worst Case: O(nlog(n))

Average Case: O(nlog(n))

Best Case: O(nlog(n))
