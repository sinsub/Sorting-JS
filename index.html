Sorting Algorithms Visualised

I have used HTML, CSS and JavaScript to visualise few of the common sorting algorithms. You can choose the input size and the type of array and hit play to see the algorithm in action.
Following are a few points explaining the visualisation:
1. Each element in the array is represented by a rectangle.
2. The value of an element is represented by the height of the rectangle
3. Different colours are used to represent pointers and sub-arrays.
4. Once the play button is clicked the array size or type can only be changed either after pausing or resetting the animation using the respective buttons, or after the array is sorted when the animation ends.
5. The reset button resets the size of the array to 10 and the type to random.
